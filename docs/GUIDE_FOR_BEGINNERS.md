## 1. このダッシュボードは何？

### ひと言で

> **「日高郡のみんな、幸せに暮らせてる？」を数字で見えるようにしたもの**です。

### もう少し詳しく

和歌山県日高郡（御坊市・日高町・みなべ町・印南町・由良町・美浜町・日高川町）の7地域に住む方々の「暮らしの満足度」を、**2つの角度**から測っています。

1. **住民の声**（アンケート調査 = 主観データ）
2. **統計データ**（国や県の公開データ = 客観データ）

この2つを組み合わせることで、「住民は満足してるけど、実はデータ上は問題がある」といった**見えにくい課題**を発見できます。

### 対象データ

| 項目 | 内容 |
|------|------|
| 回答者数 | 300人（7地域から人口比で抽出） |
| 年齢層 | 18歳〜80歳 |
| 調査年度 | FY2021〜FY2025（5年分） |
| カテゴリ | 6つのWell-Being分野 |

## 2. まず知ってほしい2つの視点：「主観」と「客観」

**これがこのダッシュボードで一番大切な考え方です。**

### 主観データとは？

> **「あなたは健康だと感じますか？」** という質問への回答

- 住民アンケートで直接聞いた「実感」「満足度」
- 1〜10点のスケール（10点満点）
- **高いほど住民が満足している**

```
例：「近所付き合いに満足していますか？」→ 7.3/10
    → 住民は「まぁまぁ満足」と感じている
```

### 客観データとは？

> **病院の数、犯罪率、バスの本数**など、実際に測れる統計数字

- 国勢調査、厚生労働省データ、県のオープンデータなどが出典
- 0〜100点に標準化（全国平均が約50点）
- **高いほど統計的に良い状態**

```
例：「公共交通のカバー率」→ 38/100
    → 全国平均（50点）より低い ＝ 交通の便が悪い
```

### なぜ両方見るの？

**片方だけでは本当の姿が見えないからです。**

| パターン | 主観 | 客観 | 何が起きている？ |
|---------|------|------|--------------|
| 理想的 | 高い | 高い | 住民も満足、データも良い |
| **要注意パターンA** | **低い** | **高い** | データは良いのに住民は不満 → 施策の成果が「伝わっていない」可能性 |
| **要注意パターンB** | **高い** | **低い** | 住民は満足しているがデータは悪い → 将来問題が顕在化するリスク |
| 課題 | 低い | 低い | 両方悪い → 早急な対応が必要 |

**パターンAの具体例：**
> 子育て支援の施設は充実しているのに（客観データ良好）、保護者の満足度は低い（主観データ低い）。
> → 施設はあるが「使いにくい」「情報が届いていない」など、別の問題がある可能性。

**パターンBの具体例：**
> 住民は「地域のつながりがある」と感じているが（主観データ良好）、統計上は自治会加入率が低下している（客観データ低い）。
> → 今は良くても、数年後にコミュニティが弱体化するリスクがある。

## 3. 6つのWell-Beingカテゴリ

「暮らしの幸福度」を6つの分野に分けて評価しています。

### カテゴリ一覧

| # | カテゴリ | 何を測っているか | 主観の質問例 | 客観の指標例 |
|---|---------|----------------|------------|------------|
| 1 | **健康・医療** | 心身の健康と医療へのアクセス | 「体調は良いですか？」「医療機関に通いやすいですか？」 | 人口あたり医師数、平均寿命、健康診断受診率 |
| 2 | **地域つながり** | ご近所や地域活動のつながり | 「近所付き合いは十分ですか？」「地域イベントに参加しますか？」 | 自治会加入率、ボランティア参加率、NPO法人数 |
| 3 | **多様性・包摂** | 多様な人が暮らしやすいか | 「地域は多様性を受け入れていますか？」 | バリアフリー施設数、外国人支援制度、文化イベント数 |
| 4 | **子育て・教育** | 子育て環境と教育の質 | 「保育施設に満足ですか？」「通学路は安全ですか？」 | 待機児童率、児童1人あたり教育費、放課後事業数 |
| 5 | **自己効力感** | 生活への満足と将来への期待 | 「生活に満足していますか？」「将来に希望がありますか？」 | 完全失業率、起業率、生涯学習講座数 |
| 6 | **交通・移動** | 移動の便利さと安全性 | 「公共交通は使いやすいですか？」 | バス路線数、道路舗装率、交通事故発生率 |

### 各カテゴリの質問構成

各カテゴリの中には**4つの細かい質問（サブ指標）**があります。

```
例：健康・医療
  ├── 身体的健康  → 「体は元気ですか？」
  ├── 精神的健康  → 「心は元気ですか？」
  ├── 医療アクセス → 「病院に行きやすいですか？」
  └── 運動習慣    → 「運動していますか？」
```

→ この4つの平均が「健康・医療」カテゴリの主観スコアになります。

## 4. ページ別ガイド：ダッシュボード

> **このページは「全体像をパッと把握する」ためのページです。**

### 4-1. 総合スコア（ゲージ）

画面上部にある大きな数字。

```
計算方法：
  300人 × 6カテゴリ × (主観 + 客観) ÷ 2
  → 全部の平均値が「総合スコア」
```

| スコア | 評価 | 意味 |
|--------|------|------|
| 8.0以上 | とても良い | 住民の暮らし全般が非常に良好 |
| 6.5〜7.9 | 良い | 概ね良好だが一部改善の余地あり |
| 5.0〜6.4 | ふつう | 平均的。積極的な施策が望まれる |
| 3.5〜4.9 | やや課題あり | 複数の分野で問題が見られる |
| 3.5未満 | 要改善 | 早急な対応が必要 |

### 4-2. レーダーチャート（六角形のグラフ）

**6カテゴリを一目で比較できるグラフ**です。

- **オレンジの形** = 主観スコア（住民の実感）
- **青の形** = 客観スコア（統計データ）

**読み方のコツ：**
- 2つの形が **重なっている** → 実感とデータが一致（良い状態）
- 2つの形が **離れている** → 乖離がある（要分析）
- ある方向だけ **凹んでいる** → その分野が弱い

```
例：交通・移動の方向だけ両方凹んでいる
  → 住民も不満、データも悪い ＝ 最優先で改善が必要
```

### 4-3. カテゴリ別スコアカード

6つのカードで各カテゴリの状態を表示。

- **枠の色**で状態がわかります
  - 青枠 → 良好（スコア6.5以上）
  - グレー枠 → ふつう（5.0〜6.4）
  - 赤枠 → 課題あり（5.0未満）

- **主観バー（オレンジ）** と **客観バー（青）** の長さの差 = ギャップ

### 4-4. 質問別スコア（カテゴリ詳細）

カテゴリを選ぶと、その中の**4つの質問ごとの平均スコア**が見られます。

- 🏆マーク = そのカテゴリで最も高いスコアの質問
- 📉マーク = 最も低いスコアの質問（改善ポイント）

### 4-5. ギャップ分析チャート

6カテゴリそれぞれの「主観 − 客観」の差をグラフ化。

```
計算方法：
  ギャップ = (主観スコア × 10) − 客観スコア

  ※ 主観（0〜10点）と客観（0〜100点）はスケールが違うので、
    主観を10倍して同じ0〜100のスケールに揃えてから比較します。
```

| ギャップ | 意味 |
|---------|------|
| +10以上 | 住民は満足しているがデータは追いついていない |
| -10以下 | データは良いが住民が不満を感じている |
| -10〜+10 | バランスが取れている |

### 4-6. デモグラフィック分析

「誰が」「どこで」幸福度が高い/低いかを分析。

- **地域別タブ**：7地域のスコア比較（どの地域が高い？低い？）
- **年代別タブ**：年齢層ごとのスコア比較（若者と高齢者で違いは？）

## 5. ページ別ガイド：客観指標

> **このページは「統計データの詳細」を確認するページです。**

### 5-1. 3つのドメイン

96個の客観指標を3つの大きなグループに分類しています。

| ドメイン | 含まれる指標数 | 具体例 |
|---------|-------------|-------|
| **生活環境** | 56指標 | 医療施設数、犯罪率、道路整備率、スーパー数 |
| **地域の人間関係** | 18指標 | 自治会加入率、NPO数、外国人比率 |
| **自分らしい生き方** | 22指標 | 投票率、失業率、平均寿命、起業率 |

### 5-2. スコアの読み方

各指標は**0〜100点に標準化**されています。

```
標準化とは？
  もともとバラバラな単位（人数、%、件数など）を、
  全国の市町村と比較して「0〜100点」に変換したもの。

  50点 ≒ 全国平均
  70点以上 ≒ 全国上位
  30点以下 ≒ 全国下位
```

### 5-3. 全国比較の見方

各指標のバーの横に表示される比較：

| 表示 | 意味 |
|------|------|
| **青いバーが長い** | 全国平均より良い |
| **オレンジのバーとほぼ同じ** | 全国平均と同水準 |
| **青いバーが短い** | 全国平均より悪い |

### 5-4. 強み指標・課題指標

- **強み指標TOP6**：日高郡が全国と比べて特に良い指標
- **課題指標TOP6**：全国と比べて特に悪い指標 → **ここが改善ポイント**

```
例：課題指標に「バス路線充足度 32/100」がある場合
  → 全国平均（50点）を大きく下回っている
  → 公共交通の改善が優先課題
```

### 5-5. 前年比の見方

各指標の隣に表示される「↑+3pt」「↓-5pt」の意味：

```
前年比 = 今年のスコア − 去年のスコア

+3pt → 去年より3ポイント改善（良くなった）
-5pt → 去年より5ポイント悪化（悪くなった）
±0pt → 変化なし
```

## 6. ページ別ガイド：予算分析

> **このページは「お金をどう使って、どんな成果が出ているか」を分析するページです。**

### 6-1. 予算の全体像

まず知ってほしい基本構造：

```
日高郡の一般会計予算（約300億円）
  └── そのうちWell-Being関連予算 → 6つのカテゴリに配分
      ├── 健康・医療    : 約32億円（最大）
      ├── 子育て・教育  : 約28億円
      ├── 交通・移動    : 約25億円
      ├── 自己効力感    : 約15億円
      ├── 地域つながり  : 約12億円
      └── 多様性・包摂  : 約8億円（最小）
```

### 6-2. 予算×成果クロスリファレンス

**このダッシュボードの中で最も重要な分析の一つ**です。

各カテゴリについて「予算を使った結果、成果は出ているか？」を判定します。

#### 予算バーの読み方

```
 ──────────████████████░░░░│░░░░░──
                           │
                    赤いマーカー ＝ 全国平均

 ████████████  ＝ 当自治体の1人あたり予算
 │             ＝ 全国平均の1人あたり予算
```

- バーが赤マーカーより**長い** → 全国平均より多く使っている
- バーが赤マーカーより**短い** → 全国平均より少ない

#### 効率スコアとは？

> **「予算1万円あたり、どれだけのスコアを得られたか」**

```
計算方法：

  主観効率 ＝ 主観スコア ÷ (予算 ÷ 1万円)
  　→ 予算1万円で住民満足度がどれだけ高いか

  客観効率 ＝ (客観スコア ÷ 10) ÷ (予算 ÷ 1万円)
  　→ 予算1万円で客観データがどれだけ良いか
  　※ 客観スコアは0-100点なので10で割って主観と同じ0-10スケールに揃えます

  総合効率 ＝ (主観スコア + 客観スコア÷10) ÷ (予算 ÷ 1万円)
  　→ 両方を合わせた総合的な費用対効果
```

**具体例で理解する：**

```
カテゴリ：地域つながり
  予算：1人あたり 1.9万円
  主観スコア：7.3/10
  客観スコア：48.8/100

  主観効率 ＝ 7.3 ÷ 1.9 ＝ 3.84（予算1万円あたり主観3.84pt）
  客観効率 ＝ 4.88 ÷ 1.9 ＝ 2.57（予算1万円あたり客観2.57pt）
  総合効率 ＝ (7.3 + 4.88) ÷ 1.9 ＝ 6.41

→ 主観効率（3.84）が客観効率（2.57）よりずっと高い！
→ 住民は満足しているけど、統計データにはまだ反映されていない
→ 「ソフト事業（イベント・相談窓口等）は効果的だが、
    ハード面（施設整備等）の投資が足りない可能性」
```

#### 全国基準との比較

```
全国基準効率の考え方：
  「もし全国平均の予算で全国平均のスコアを達成したら、
   効率はいくつになるか」を基準値にしています。

  全国基準効率 ＝ (主観6.0 + 客観5.0) ÷ (全国平均予算÷1万円)

  ▲+15% → 全国基準より15%効率が良い（少ない予算で成果を出している）
  ▼-10% → 全国基準より10%効率が悪い（予算のわりに成果が出ていない）
```

#### 主観効率と客観効率の乖離が教えてくれること

| 主観効率 | 客観効率 | 何を意味するか |
|---------|---------|-------------|
| **高い** | **高い** | 予算が両方に効いている（理想的） |
| **高い** | **低い** | 住民は満足だが統計には未反映 → ハード投資不足？ |
| **低い** | **高い** | データは改善しているが住民に伝わっていない → 広報不足？ |
| **低い** | **低い** | 予算が効果を発揮していない → 事業の見直しが必要 |

### 6-3. カテゴリ別パイプラインカード

各カテゴリの「お金の流れ → 成果」を1枚のカードにまとめたもの。

```
┌─────────────────────────────────────────┐
│ [アイコン] カテゴリ名        [効率バッジ] │
│                                         │
│  予算       ████████████  執行率 86%     │
│  8.8億      前年比 +7.9%                │
│                                         │
│  ┌ 主観・客観の比較         乖離 1.2pt ┐ │
│  │ 主観 ████████░░░░  5.0             │ │
│  │ 客観 █████████░░░  4.9             │ │
│  └────────────────────────────────────┘ │
│                                         │
│  ▼ 5事業の内訳を見る                     │
└─────────────────────────────────────────┘
```

**読み方：**
1. **予算と執行率**：予算をどれだけ実際に使ったか
   - 85%以上 = 緑（順調に使えている）
   - 75-84% = オレンジ（やや遅れ）
   - 75%未満 = 赤（予算が使い切れていない → 事業が計画通り進んでいない）

2. **主観・客観の比較**：2つのバーで直感的に比較
   - バーの長さの差 = ギャップ

3. **事業の内訳**：クリックすると個別事業の執行状況が見られる

### 6-4. 歳出構成ドーナツグラフ

一般会計予算の使い道を円グラフで表示。

```
例：
  民生費（福祉）  35%  ← 最大
  教育費          18%
  土木費          12%
  衛生費          10%
  ...
```

この中で「Well-Beingに関連する部分」がどれくらいかを**WB比率**として表示しています。

### 6-5. 年度別トレンド

FY2021〜FY2025の5年間で、各カテゴリの予算がどう推移したかを折れ線グラフで表示。

```
見るポイント：
  ・右肩上がり → 予算が増加（重点化されている）
  ・横ばい     → 変化なし
  ・右肩下がり → 予算削減（注意が必要）
```

### 6-6. 執行率とは

> **「もらった予算をどれだけ実際に使えたか」の割合**

```
執行率 ＝ 実際に使った金額 ÷ もらった予算額 × 100%

例：予算10億円で、実際に使ったのが8.5億円
  → 執行率 85%
```

**なぜ100%じゃないの？**
- 事業の遅れ（入札不調、人手不足など）
- 想定より安く済んだ
- 事業そのものが計画通り進まなかった

**低い執行率が示す問題：**
- 75%未満 → 予算は確保したが事業が進んでいない
- → 人員不足？制度設計に問題？住民ニーズとのミスマッチ？

## 7. ページ別ガイド：インサイト

> **このページは「データから読み取れる重要な発見」をまとめたページです。**

### 7-1. インサイトとは？

> **データを分析して自動的に見つけた「注目すべきポイント」**

人間が手作業で分析すると見落としがちなパターンを、あらかじめ設定した条件に基づいて自動検出しています。

### 7-2. 6つのインサイトタイプ

#### タイプ1: 高予算・低成果（重大）🔴

```
検出条件：
  1人あたり予算が 1.5万円超  かつ  客観スコアが 50点未満

意味：
  「たくさんお金を使っているのに、成果が出ていない」
  → 最優先で原因を調べて対策を打つべき
```

#### タイプ2: 低予算・高成果（機会）💡

```
検出条件：
  1人あたり予算が 1.2万円未満  かつ  客観スコアが 55点超

意味：
  「少ない予算で高い成果を出している」
  → 成功事例として他のカテゴリにノウハウを横展開できる
```

#### タイプ3: 主観-客観ギャップ（警告）⚠️

```
検出条件：
  |ギャップ| > 12ポイント

意味：
  「住民の実感と統計データが大きくズレている」

  ギャップがマイナス → データは良いのに住民は不満
    → 施策の成果を住民に伝える広報が足りない？

  ギャップがプラス → 住民は満足だがデータは悪い
    → 今は良くても将来問題が出てくるリスクあり
```

#### タイプ4: 効率ギャップ（警告）⚠️

```
検出条件：
  主観効率 ÷ 客観効率 > 1.5倍  または  < 0.67倍

意味：
  「予算が主観と客観に偏って効いている」

  主観効率 >> 客観効率 の場合：
    → 住民の満足度は上がっているが、統計データは改善していない
    → ソフト事業（相談・イベント等）は効果的だが、
      ハード事業（施設・インフラ等）が不足している可能性

  客観効率 >> 主観効率 の場合：
    → 統計データは改善しているが、住民は気づいていない
    → 成果の広報・見える化が不足している可能性
```

#### タイプ5: トレンドアラート（警告）⚠️

```
検出条件：
  前年比で 8ポイント以上低下した客観指標がある

意味：
  「急激に悪化している指標がある」
  → 外部環境の変化か、内部の問題かを早急に切り分ける必要あり
```

#### タイプ6: 低執行率アラート（警告）⚠️

```
検出条件：
  執行率75%未満の事業がある

意味：
  「予算が計画通りに使えていない事業がある」
  → 人員不足？制度設計の問題？対象者が少ない？
```

## 8. よくある疑問Q&A

### Q1: 「スコア6.5」って良いの？悪いの？

**A:** 6.5は「良い」に分類されます。目安は以下の通り：

| 範囲 | 評価 | イメージ |
|------|------|---------|
| 8〜10 | とても良い | クラスでトップレベル |
| 6.5〜8 | 良い | 平均以上。もう一歩で「とても良い」 |
| 5〜6.5 | ふつう | 可もなく不可もなく |
| 3.5〜5 | やや課題 | 改善が必要 |
| 1〜3.5 | 要改善 | 早急な対応が必要 |

### Q2: なぜ主観と客観でスコアの単位が違うの？

**A:** 主観（1〜10点）と客観（0〜100点）は、元のデータの性質が全く違うためです。

- **主観**：「1=とても不満 〜 10=とても満足」のアンケート回答
- **客観**：さまざまな統計を全国比較で標準化した値

比較するときは、主観を10倍（× 10）して0〜100スケールに揃えてから差を取ります。

### Q3: 「効率スコア5.0」の意味は？

**A:** 「予算1万円あたり、主観+客観の合計で5.0ポイント分の成果を生んでいる」という意味です。

- 全国基準より高い → 効率的にお金を使えている
- 全国基準より低い → 予算に見合う成果が出ていない

ただし、これは**あくまで参考値**です。たとえば災害復旧など、すぐには成果が出ない投資もあります。

### Q4: ギャップがあるのは悪いこと？

**A:** 必ずしも悪いことではありません。

- **短期的なギャップ**は自然に発生します（施策を打ってもデータに反映されるまで時間がかかる）
- **長期的・大きなギャップ**は問題の兆候です

目安として：
- ±5pt以内 → 正常範囲
- ±5〜12pt → 注視が必要
- ±12pt超 → インサイトとして自動検出される

### Q5: 全国平均って本当に「平均的」？

**A:** 客観指標の全国平均（50点）は、全国約800の市町村の中央値付近です。つまり：

- 50点超 → 全国の半分以上の自治体より良い
- 50点未満 → 全国の半分以上の自治体より悪い

ただし、日高郡は地方・郡部なので、都市部と同じ基準で比較すると厳しい面もあります。特に「交通」は都市部と比べると構造的に不利です。

### Q6: 予算を増やせばスコアは上がるの？

**A:** 単純に「予算を増やす＝スコアが上がる」とは限りません。

このダッシュボードが示しているのは：
1. **効率の良い使い方**をしている分野はどこか
2. **予算を使い切れていない**事業はないか
3. **住民のニーズと施策のズレ**はないか

大切なのは「いくら使うか」ではなく「どう使うか」です。
